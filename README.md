1. Create the folder (if it doesn't exist): `C:\Users\USER_NAME\AppData\Local\UnrealEngine\Slate\Themes\`
2. Download the .json from the repo
3. Open Unreal Editor -> Edit -> Editor Preferences -> Active Theme -> Import a theme from a file -> Select the json
